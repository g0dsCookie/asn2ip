This is a simple tool to pull all netblocks from an ASN into a text file output.
You can use this in your application (e.g. firewall like pfSense/OPNsense) to filter or prioritize specific services.

How to use

To use this service just send a GET request with the ASN number as path.
You may also request multiple ASN by seperating them with a ':'.

Examples:
Netflix https://asn.copr.icu/2906
Netflix and Twitch https://asn.copr.icu/2906:46489

Options

You can use the following options in your GET request to control the output of this tool.

Option Type Description Default
ipv4 Boolean (y, yes, true) Output IPv4 addresses. true
ipv6 Boolean (y, yes, true) Output IPv6 addresses. true
separator String Use this as the separator between IP-Addresses [[:space:]]

You can also request a json output by setting the Accept header to application/json.

Examples:
Netflix IPv4 only https://asn.copr.icu/2906?ipv4=true&ipv6=false
Twitch as JSON https://asn.copr.icu/46489?json=yes
Twitch comma seperated https://asn.copr.icu/46489?seperator=%2C
